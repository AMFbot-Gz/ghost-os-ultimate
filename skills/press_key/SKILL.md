# press_key

Appuie sur une touche via AppleScript `System Events`.

## Params

- `key` (string, optionnel) : touche à presser. Défaut: "Return". Valeurs: Return, Space, Tab, Escape, Backspace, ArrowUp, ArrowDown, ArrowLeft, ArrowRight
